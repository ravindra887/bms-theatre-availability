BMS Alerts : BMS Search API

Created by ravindra Varma, last modified on Nov 25, 2022

BMS OpenAPI for searching movies and theatres.

Request:

curl 'https://in.bookmyshow.com/pwa/api/search/quick-search' \
  -H 'authority: in.bookmyshow.com' \
  -H 'cache-control: no-cache' \
  -H 'content-type: application/json' \
  -H 'origin: https://in.bookmyshow.com' \
  -H 'pragma: no-cache' \
  --data-raw '{"searchText":"avatar","isSubscribed":"N","regionCode":"HYD","userEmail":"","lat":"","lon":"","strRef":false}' \
  --compressed
